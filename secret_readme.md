## Installation

Prerequisite: having [Ninja](https://ninja-build.org/) installed (version 1.7+), e.g. `brew install ninja` if you're on macOS.

```sh
git clone https://github.com/reasonml/rehydrate.git
cd rehydrate
npm install
npm start
# in another window
npm run build
```
